## 🧠 Sobre este repositorio

Este repositorio contiene una recopilación de proyectos que he desarrollado mientras seguía el curso de React de **midudev** en YouTube.

⚠️ **Importante:**  
Los proyectos son réplicas educativas basadas en las clases del curso, con el objetivo de aprender y reforzar conceptos.

## 🚀 Objetivos

- Aprender React desde cero hasta nivel intermedio/avanzado  
- Entender cómo estructurar aplicaciones frontend  
- Practicar buenas prácticas en desarrollo web  
- Mejorar lógica y resolución de problemas
